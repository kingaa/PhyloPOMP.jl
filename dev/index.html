PhyloPOMP.scan_branch! — Method
scan_branch!(input, G, p, mapper)

Parse the branch-string in input, appending the corresponding node to Genealogy G.

Arguments:

  • input: the string, or vector of strings, containing the branch information
  • G: the genealogy to be modified
  • p: the name of the parent node
  • mapper: a Dict mapping the deme-specification string to the correct deme.

PhyloPOMP.Genealogy — Type
Genealogy{D}

Implements a genealogy over a time interval [t0,t]. The Enum-type D enumerates the demes. Internally, this is represented as a time-ordered sequence of genealogical nodes (represented by GenealNode objects).
