# PhyloPOMP.jl

For doing phylodynamics in Julia.
**PhyloPOMP.jl** enables computation of the phylodynamic likelihood using tools from [POMP.jl](https://github.com/kingaa/PartiallyObservedMarkovProcesses.jl/).
